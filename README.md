# E2E тест для saucedemo.com

Проект содержит автоматический e2e тест для проверки сценария покупки товара на сайте saucedemo.com.
Реализованы автоматизированные тесты с использованием шаблона Page Object Model (POM), который организует взаимодействия веб-страниц в отдельные классы, каждый из которых представляет страницу или функцию.

## Инструкция по запуску

1. Установите зависимости:

    ```sh
    pip install selenium
    pip install pytest
    ```

2. Запустите тест:

    ```sh
    pytest test_cart.py
